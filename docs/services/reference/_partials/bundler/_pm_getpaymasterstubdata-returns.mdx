An object containing stub paymaster-related fields for gas estimation.

<Tabs>
<TabItem value="v0.7" label="v0.7 and v0.8">

- __INLINE_CODE_0__: (string) - The address of the paymaster contract.
- __INLINE_CODE_1__: (string) - The data to be used in the __INLINE_CODE_2__ field of the user operation.
- __INLINE_CODE_3__: (string) - The amount of gas to allocate for the verification step
    of the paymaster.
- __INLINE_CODE_4__: (string) - The amount of gas to allocate for the post-operation step of
    the paymaster.

</TabItem>
<TabItem value="v0.6" label="v0.6">

- __INLINE_CODE_5__: (string) - The combined paymaster address and data to be used in the user operation.

</TabItem>
</Tabs>
